# vue_app
项目概述: 综合品类平台购物商城的前台PC端。主要实现的内容有首页、商品详情、购物车、支付、个人中心、订单管理、登录。商城前台的设计与实现，该商城提供了两个登录角色：游客和用户。游客可以不用登录直接
浏览商品，加入购物车，用户则可以购买商品、修改购物车以及进入个人中心。 

技术选型: Vue2+Vuex+Axios+elementUI+Mock+Swiper
